Free Test Database

Visit this link to try out a free database. It is formatted as an addressbook, so go ahead and add your name once you finish exploring this website. This will show you how databases work as it has column and rows as a proper database does. It is a common example of a relational database. Also, it demonstrates permissions as you have SELECT, INSERT, UPDATE, DELETE, CREATE priviliges.

Free MySQL Hosting

If you would like to have an external database for testing, go to Remote MySQL. I have used them before on multiple occasions and found their databases very reliable. They use randomized user and password combos for security but you can change those in PHP WebAdmin. However, only use this service if you know that you will not need more than 1 GB of database storage.
